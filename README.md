# Luri (SLB) Alphabet to IPA Mapping

## Vowels

| SLB | IPA | Type | Example |
|-----|-----|------|---------|
| a | [ɑ] | full | máta |
| á | [ɐ] | reduced | Fásl |
| e | [e] | full | Ebrahim, Eshaq |
| é | [ə] | reduced (schwa) | béřám |
| i | [i] | full | Isa |
| o | [o] | full | kor |
| u | [u] | full | Yusof |
| ä | [ɐː] | long á | Yäqub |
| ë | [eː] | long e | mëtár |
| ü | [uː] | long u | Büáz |
| ö | [oː] | long o | göt |
| î | [ʔi] | diphthong (glottal stop + i) | tábîd |

**Note:** Acute accent (á, é) marks vowel reduction.
